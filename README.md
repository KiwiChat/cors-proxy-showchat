# ro-proxy-cors

RO Proxy CORS
Activați CORS (partajarea resurselor încrucișate) pentru orice adresă URL cu această API simplă și converentă!

Proxy invers cu anteturile CORS.

# Utilizare:

Efectuați o operațiune de mai jos:
```
https://showchat.gq/<url>
```
